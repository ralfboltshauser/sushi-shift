# Sushi Shift

Sushi Shift is a Godot 4.6 sushi cooking game prototype.

## Controls

- `WASD`: move
- `E`: interact
- `Q`: drop item
- `Esc`: pause

## Run Locally

Open this folder in Godot 4.6.3 or newer and run `scenes/Main.tscn`.

## Export

Configured export presets:

- `Linux`: `builds/linux/SushiShift.x86_64`
- `macOS`: `builds/macos/SushiShift-macOS.zip`
- `Windows`: `builds/windows/SushiShift.exe`
